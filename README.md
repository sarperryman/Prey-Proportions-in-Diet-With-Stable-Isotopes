# Prey-Proportions-in-Diet-With-Stable-Isotopes

The two needed datasets are "CARMMHA_Mixing.Model_2020.05.04" and "CARMMHA_Stable.isotope.ratios_2021.01.20" and the XGBoost code is "XGBoostCode"

**Motivation**

This project examines whether XGBoost can use stable isotope data to estimate the contribution of different prey groups to bottlenose dolphin diets. Stable isotope analysis is commonly used in marine food web studies because carbon isotopes (δ¹³C) provide information about foraging habitat, while nitrogen isotopes (δ¹⁵N) provide information about trophic position. Traditionally, Bayesian mixing models are used to estimate diet proportions by comparing predator isotope values to those of potential prey groups while accounting for trophic discrimination factors. In this project, Bayesian mixing-model outputs are used as reference values to test whether an XGBoost machine learning model can approximate diet proportions using predator and prey stable isotope values as inputs.

**Data**

The data used in this project comes from the Gulf of Mexico Research Initiative Information and Data Cooperative (GRIIDC) CARMMHA Stable Isotope Dataset and previously generated Bayesian mixing-model outputs (https://data.griidc.org/data/R6.x809.000:0020). The dataset includes δ¹³C and δ¹⁵N stable isotope values from bottlenose dolphins and potential prey species collected across several Gulf of Mexico locations, including Mobile Bay, Barataria Bay, and Sarasota Bay. The data were cleaned and standardized by combining predator, prey, and mixing-model datasets, converting isotope values into numeric format, and grouping samples by location and year to calculate average isotope values. These averaged isotope values were then merged with Bayesian mixing-model diet proportion estimates to create the final dataset used for the XGBoost analysis.

**Model**

To estimate dolphin diet composition, this project used an XGBoost regression model. The model used predator and prey stable isotope values, including δ¹³C and δ¹⁵N measurements, as predictor variables, while Bayesian mixing-model diet proportions served as the target outputs. XGBoost was selected because relationships between isotope values and diet composition are likely complex and nonlinear, particularly when prey groups have overlapping isotope signatures. The model can also handle interactions among variables and missing predictor values more effectively than many traditional statistical approaches. Because the final dataset was relatively small after combining isotope and mixing-model data across locations and prey groups, the data could not be reliably divided into separate training and testing datasets. Instead, model performance was evaluated by comparing predicted diet proportions directly to the original Bayesian mixing-model estimates using R² and root mean square error (RMSE).

**Analysis**

If the XGBoost model predicts diet proportions with relatively strong agreement with the Bayesian mixing-model outputs, this suggests that stable isotope values contain ecological information related to dolphin diet composition. Some prey groups may be predicted more accurately than others because broader prey categories, such as total fish contribution, may occupy more distinct regions of isotopic space than narrower prey groups with overlapping isotope signatures. If model performance is weak, this may indicate that stable isotope values alone do not contain enough information to reliably predict detailed diet proportions for all prey groups. Poor performance may also reflect overlap among prey isotope signatures, small sample size, missing prey data, and regional differences in isotopic baselines across Gulf of Mexico locations. Overall, these results help evaluate the strengths and limitations of using machine learning approaches to approximate Bayesian stable isotope mixing-model outputs from isotope data alone.
